# actguard

SDKs for the actguard platform.

## Repository Structure

```
actguard/
├── docs/           # Documentation
├── examples/       # Usage examples
└── libs/
    ├── sdk-py/     # Python SDK
    └── sdk-js/     # JavaScript/Node.js SDK
```

## SDKs

### Python SDK

Located in `libs/sdk-py/`. Published to [PyPI](https://pypi.org/) via Trusted Publisher Management.

### JavaScript SDK

Located in `libs/sdk-js/`.

## Development

See the individual SDK directories for setup and development instructions.
